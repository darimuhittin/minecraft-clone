## Technologies

- GLM (OpenGL Mathematics)

## Building

### Prerequisites

- CMake (3.15 or higher)
- C++17 compatible compiler
- vcpkg package manager

### Dependencies

- GLFW3
- GLM
- OpenGL

### Build Instructions

```bash
mkdir build
cd build
cmake ..
cmake --build .
```
